
##Task Manager App

Dies ist eine Task-Manager-App, die entwickelt wurde, um Aufgaben und Termine zu verwalten. Das App wurde mit Flutter
programmiert und verwendet SQLite als Datenbank zur Speicherung von Aufgaben und Terminen.
 

##Features:
local notification,
local database,
dark theme,


## Functions:

- Hinzufügen, Bearbeiten und Löschen von Aufgaben und Terminen.
- Einstellen von Erinnerungen für Aufgaben und Termine.
- Unterstützung für wiederkehrende Aufgaben oder Termine.
- Anpassbare Wochenansicht zur übersichtlichen Darstellung der Aufgaben und Termine.
- Benachrichtigungen für anstehende Aufgaben und Termine.


## Installation

1. Stellen Sie sicher, dass Flutter auf Ihrem System installiert ist. Weitere Informationen finden Sie in der offiziellen [Flutter-Dokumentation](https://flutter.dev/docs/get-started/install).

2. Klonen Sie dieses Repository auf Ihren lokalen Computer: https://git.imp.fu-berlin.de/redwaa99/task-manager-app.git, Navigieren Sie in das Projektverzeichnis: cd /your/path/to/the/local/repository. 
3. Führen Sie den folgenden Befehl aus, um die erforderlichen Abhängigkeiten zu installieren: flutter pub get.

4. Starten Sie die App, indem Sie den folgenden Befehl ausführen: flutter run

## Konfiguration

Die App verwendet SQLite als Datenbank zur Speicherung von Aufgaben und Terminen. Die SQLite-Konfiguration ist bereits in das Projekt integriert, daher ist keine zusätzliche Konfiguration erforderlich.

## Libraries, Tools & Packages Used:
- Flutter SDK - Das UI-Toolkit und Framework zur Entwicklung von plattformübergreifenden mobilen Apps.
- SQLite - Eine leichte und schnelle lokale Datenbank zur Speicherung von Aufgaben und Terminen.
- Provider - Ein State-Management-Package für Flutter, das die Verwaltung des App-Zustands erleichtert.
- Flutter Local Notifications - Ein Package, um lokale Benachrichtigungen für Aufgaben und Termine anzuzeigen.
- Flutter DateTimePicker - Ein Package, um einen interaktiven Datums- und Zeitpicker in der App zu implementieren.
- Get - Eine State-Management- und Routing-Bibliothek für Flutter.
